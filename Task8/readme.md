# To-Do Web App

A simple and interactive to-do list web application that allows users to add, complete, and delete tasks efficiently.

## Features

- ✅ **Add Tasks**: Enter a task and click the "Add" button or press Enter
- ✅ **Mark Complete**: Click on any task to toggle its completion status (strikethrough text)
- ✅ **Delete Tasks**: Click the "✖" button next to a task to remove it
- ✅ **Input Validation**: Prevents empty tasks from being added
- ✅ **Responsive Design**: Clean and user-friendly interface with hover effects

## Files

- **index.html**: Main HTML structure of the application
- **style.css**: Styling for the to-do app with a modern look
- **script.js**: JavaScript functionality for adding, rendering, and managing tasks

## How to Use

1. Open `index.html` in your web browser
2. Type your task in the input field
3. Click the "Add" button or press Enter to add the task
4. Click on a task to mark it as completed (strikethrough effect)
5. Click the "✖" symbol to delete a task

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
