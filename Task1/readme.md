# Personal Profile Page

This is a personal profile webpage created using semantic HTML tags.

## Description
The page includes personal information such as name, bio, education, skills, and contact details. Semantic HTML elements are used to improve accessibility and page structure.

## Features
- Semantic HTML elements (`header`, `nav`, `section`, `article`, `footer`)
- Internal navigation using anchor links
- Profile image with accessible `alt` text
- Clean and readable structure

## Technologies Used
- HTML5

## How to Run
Open the HTML file in any modern web browser.
